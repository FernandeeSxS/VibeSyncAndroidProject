# VibeSync

VibeSync é uma aplicação Android que permite aos utilizadores gerir playlists. Desenvolvida em Kotlin com Jetpack Compose, integra Firebase para autenticação e armazenamento de dados, oferecendo uma experiência completa e moderna para amantes da música.

## Funcionalidades

- Autenticação de utilizadores com email e password
- Registo e login com visualização da password opcional
- Criação, edição e eliminação de playlists
- Adição, edição e eliminação de músicas em playlists
- Visualização de detalhes das músicas
- Interface interativa e intuitiva com Jetpack Compose
- Design responsivo e moderno

## Tecnologias

- Kotlin
- Jetpack Compose
- Firebase (Auth e Firestore)
- Android Studio
- Git para controlo de versão
